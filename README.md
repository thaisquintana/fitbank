# Fitbank - Test

## Setup

```
- node 8 or later
- npm 6
- npm install -g sass [Sass](https://sass-lang.com/install)
```

## Development

```
$ git clone git@github.com:thaisquintana/fitbank.git
$ cd fitbank
$ npm install
```

## Web

To run application:
-According to documentation ([Create React App])

- [npm start](#npm-start)
- [npm run build](#npm-run-build)

## Local Settings

#### Eslint in Visual Studio Code

- Install extension `ESLint`
- Install extension `Prettier - Code formatter`
  Edit `settings.json` in VS Code (file > preferences > settings)

```
{

  "prettier.eslintIntegration": true, //Adjust rules to eslint
  "editor.formatOnSave": true, //Automatic corrections when saving file

}
```

#### EditorConfig in Visual Studio Code

- Install the extension in VS Code `EditorConfig for VS Code`
